# Definition of Done (DoD)

- Code qua lint `flutter analyze`; không có lỗi mức error.
- UI chạy ở theme sáng/tối; tương phản đạt yêu cầu.
- i18n: khoá vi/en đầy đủ; `flutter gen-l10n` chạy ok.
- Luồng tính năng có test cơ bản (unit/widget) hoặc script kiểm thử tay kèm bằng chứng (video).
- Không lộ secrets; cấu hình AI qua `--dart-define` khi cần.
- Firestore: thao tác CRUD đúng collection; trường bắt buộc có dữ liệu.
- Tài liệu cập nhật: backlog, sprint plan, HLD, changelog.
